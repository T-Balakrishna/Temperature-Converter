# Temperature Converter

Live Demo: [https://temperature--converter.vercel.app/](https://temperature--converter.vercel.app/)

## Description

A simple and efficient web application for converting temperatures between Celsius and Fahrenheit. Built with a focus on clarity, responsiveness, and fast performance.

## Features

* Real-time conversion between Celsius and Fahrenheit
* Responsive layout for desktop and mobile devices
* Clean and minimal user interface
* Lightweight and fast build

## Installation

```bash
git clone https://github.com/T-Balakrishna/Temperature-Converter.git
cd temperature-converter
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Usage

Enter a value in either temperature field to see the converted result instantly. Use the reset option to clear inputs.
